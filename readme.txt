Allow users with certain emails to register to your WordPress site by adding whitelist rules.

== Description ==
- Allow certain kinds of users to register to your WordPress site by adding whitelist rules. 
- Super simple to use. The whole plugin just has one simple settings page.
- Support regular expression rules.
- Completely free.

== Installation ==
- Download the plugin package.
- Copy it to wp-content/plugins
- Unzip the package.
- Activate it in your WordPress dashboard
- Configure in 'Geek Mail Whitelist' page.
